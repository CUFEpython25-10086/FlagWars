FlagWars 是一款现代化的*夺旗*游戏。在这个游戏中，每个玩家都有一个基地。游戏的目标是占领对手的基地，成为最后一个拥有基地的玩家。

## 地形类型
游戏中有多种地形类型，包括：

1. **平原**：平原是最基本的地形类型。它是平坦且可通行的。你占领的每个平原地块每15个游戏刻会生成一个士兵。

2. **基地**：基地是玩家的家园。它是可通行的，并且每个游戏刻都会生成一个士兵。如果一个玩家的基地被占领，该玩家将输掉游戏。

3. **塔楼**：塔楼是可通行的地形。占领塔楼需要特定数量的士兵。如果玩家占领了塔楼，塔楼将像基地一样每个游戏刻生成一个士兵。塔楼在游戏中自然随机生成。

4. **城墙**：城墙是不可通行的地形。它可以被特定数量的士兵摧毁。城墙在游戏中自然随机生成。

5. **山脉**：山脉是不可通行的地形。它在游戏中自然随机生成。与城墙不同，山脉无法被摧毁。

6. **沼泽**：沼泽是可通行的地形。它在游戏中自然随机生成。沼泽会减少站在其上的士兵，每个游戏刻减少一个士兵。

## 游戏玩法
玩家可以使用鼠标指针选择自己的士兵。被选中的士兵将会高亮显示。使用方向键来规划士兵的移动。

当玩家的士兵踏上某个地块时，该地块就被占领了。多个玩家的士兵在同一地块上会相互抵消。某些地形类型，如城墙和塔楼，会抵消地块上的士兵，并减少占领所需士兵数量，直到被完全占领。

当除了一个玩家之外的所有玩家都失去了他们的基地时，游戏结束。

进攻、防守、合作与背叛……利用一切机会赢得胜利！

# 技术栈

- **服务端**：Python + Tornado（WebSocket通信）
- **客户端**：HTML5 Canvas + Brython（Python in Browser）
- **包管理**：uv（现代Python包管理器）
